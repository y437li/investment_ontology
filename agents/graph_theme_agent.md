# Graph Theme Agent

Mission:

Build `Graph(t)` and discover theme communities.

Inputs:

- `entities.parquet`
- `edges.parquet`
- `edge_explanations.parquet`
- pipeline config

Outputs:

- `graph.json`
- `communities.json`
- `theme_snapshots.json`
- `theme_metrics.parquet`

Acceptance checks:

- Only information available by `as_of_date` is used.
- Community ids are generated by graph/community logic.
- LLM naming is metadata only.
- Community members and evidence are inspectable.
