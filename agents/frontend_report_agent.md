# Frontend Report Agent

Mission:

Make artifacts inspectable and produce evidence-backed reports.

Inputs:

- all run artifacts.
- theme and validation outputs.

Outputs:

- dashboard views.
- `report.md`.

Acceptance checks:

- Every major report claim points to evidence, graph, exposure, or validation artifacts.
- The UI supports evidence drilldown.
- The report does not make unsupported prediction claims.
