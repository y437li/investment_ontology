# Skill: Evidence Report Generation

Purpose:

Generate a research report from artifacts without inventing conclusions.

Inputs:

- theme snapshots.
- company exposure.
- validation results.
- edge explanations.

Steps:

1. Summarize emerging and accelerating themes.
2. List top exposed companies.
3. Cite evidence chunks and graph artifacts.
4. Cite validation results.
5. Add caveats and failure cases.

Outputs:

- `report.md`

Failure modes:

- Unsupported investment claims.
- Claims without evidence.
- Confusing theme names with discovered communities.
